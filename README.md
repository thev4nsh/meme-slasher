# 🎮 Meme Slasher

A fast-paced Flutter game with Firebase integration, leaderboard system, and in-app update functionality.

---

## 🚀 Features

* 🎮 Smooth gameplay
* 🏆 Leaderboard system
* 🔔 Push notifications (Firebase)
* 📲 In-app update system
* 🎥 Tutorial guide

---

## 🛠️ Tech Stack

* Flutter (Dart)
* Firebase (Firestore + FCM)
* Provider (State Management)
